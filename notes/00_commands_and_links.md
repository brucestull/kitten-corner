# Commands & Links

## Commands

- Create virtual environment and install dependencies:
    - `pipenv install django black isort flake8`

- Build, in `kitten-corner`:
    - `python setup.py sdist`
- Install, in `kats`:
    - `pipenv install ..\kitten-corner\dist\KittenCorner-0.1.tar.gz`
